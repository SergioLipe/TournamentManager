# Torneio

A simple web application built with PHP and MySQL to create and manage tournaments.

## Features

- Create tournaments with predefined themes or create your own.
- Perfect for decision-making or fun competitions with friends.
- Choose between 2 to 16 images per theme.
- Optionally use names instead of images.

## Technologies Used

- PHP
- MySQL

## Usage

1. Set up your MySQL database.
2. Configure your environment variables (`.env`).
3. Launch the web server.
